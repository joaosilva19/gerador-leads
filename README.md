# Gerador de Leads com Streamlit

Este aplicativo permite gerar leads de empresas e responsáveis com base em filtros como cidade, setor, cargo e número de funcionários.

## Como usar

1. Faça o clone do repositório:
   ```bash
   git clone https://github.com/SEU_USUARIO/gerador-leads.git
   cd gerador-leads
   ```

2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```

3. Rode o aplicativo:
   ```bash
   streamlit run main.py
   ```

Ou publique automaticamente no [Streamlit Cloud](https://streamlit.io/cloud).

## Filtros incluídos

- Cidade
- Setor
- Cargo
- Número de Funcionários
